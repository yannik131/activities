{% blocktrans %}Herzlich willkommen auf myactivities.net. Sieht alles noch ein wenig rustikal aus, funktioniert aber. Darum geht's hier: Sagen wir, Sie spielen gerne Schach und Gitarre. Sie suchen einen Gegner und eine Band, möglichst in Ihrer Stadt. Dann tun Sie folgendes:{% endblocktrans %}

  1. {% blocktrans %}Sie registrieren sich und geben die Stadt an, in der Sie wohnen.{% endblocktrans %}
  2. {% blocktrans %}Einmal eingeloggt, können Sie unter "Aktivitäten" die Aktivität "Schach spielen" auswählen und ihr beitreten.{% endblocktrans %}
  3. {% blocktrans %}Über die Mitgliederliste finden Sie Menschen in Ihrer Stadt/Ihrem Landkreis/Ihrem Bundesland, die auch gerne Schach spielen. Falls Sie mit jemandem Kontakt aufnehmen wollen, schicken Sie dieser Person einfach eine Freundschaftsanfrage.{% endblocktrans %}
  4. {% blocktrans %}Unter "Matches und Turniere" finden Sie zurzeit in Ihrer Gegend stattfindende Veranstaltungen, auf die Sie sich bewerben können. Sie können auch ganz einfach eigene Matches und Turniere erstellen. In einem Turnier werden nach dem Schweizer System Spielpläne (Runden) und eine Punktetabelle mit Tie-Breaks generiert (vom Spiel abhängig).{% endblocktrans %}
  5. {% blocktrans %}Die Aktivität "Gitarre spielen" liegt in der Kategorie "Musik". Auf der Kategorieseite können Sie nach Gruppen in Ihrer Umgebung suchen, die offene Leerstellen haben. Sie können auch selbst eine Gruppe gründen und so nach Musikern suchen.{% endblocktrans %}
  6. {% blocktrans %}In einer Gruppe können Sie Terminabstimmungen für ein Treffen erstellen. Suchen Sie nach Mitgliedern, erzeugen Sie einfach eine Leerstelle und geben dort Bewerbungskriterien wie Alter, Wohnort oder Geschlecht an, z. B.: Bassist gesucht, Alter zwischen 20-30, Geschlecht egal, Wohnort Oldenburg.{% endblocktrans %}
  7. {% blocktrans %}Jede Gruppe und jeder Nutzer kann in einer Kategorie bzw. einer Aktivität etwas posten. Die Posts sind über die jeweilige Aktivitäts- bzw. Gruppenseite zu sehen und werden dort nach Stadt/Landkreis/Bundesland gefiltert. Natürlich sind die Posts auch über die Profilseite des Nutzers bzw. der Gruppe erreichbar.{% endblocktrans %}
  8. {% blocktrans %}Für jede Freundschaft, jede Gruppe, jede Bewerbung, jedes Match und jedes Turnier gibt es einen eigenen Chat.{% endblocktrans %}
  9. {% blocktrans %}Gruppen können in jeder Kategorie gegründet werden. Matches und Turniere stehen nur bei wettkampffähigen Kategorien zur Verfügung.{% endblocktrans %}
